To build PDK and HDK docs do the following:

1. Edit cpemdocs.scp and packemdocs.scp and set the base path for the EMSDIO project and output directories.
2. Run cpemdocs.scp in a CYGWIN or BASH shell
3. Run the CreateDocs and MakeLinks wizards for PDK and HDK
4. Run packemdocs.scp in a CYGWIN or BASH shell.
