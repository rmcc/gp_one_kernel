Embedded SDIO Stack Overview

Codetelligence’s Embedded SDIO technology offering consists of a collection of system software components and tools to support SD Memory and SDIO card peripherals on embedded systems. The system software components are based on Codetelligence’s proprietary Embedded SDIO stack architecture. These components provide a hardware abstraction and card function API layer to allow OEM, IHV or ISV supplied software to communicate and control cards across an SD/SDIO bus. Codetelligence provides the necessary adaptation layers to allow the Embedded SDIO stack to operate under a variety of embedded operating systems. This platform solution allows a device manufacturer to rapidly include SD/SDIO support in their product with less cost and risk.
