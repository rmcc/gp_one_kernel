SDIO Host Controller Driver Quick Start

The developer should first become familiar with the driver model for the target operating system. This manual provides only a brief overview of the driver development and debugging processes.

We encourage you to read the section: Developing Host Controller Drivers. This SDK is supplied with one or more driver samples in the install tree located at \hcd. This directory includes a PCI-based, SDIO Standard Host Controller driver sample. Some version of this kit may include samples for controllers on various System On Chip devices (i.e. PXA25x). The sample drivers demonstrate:

Host Controller registration/deregistration.

Handling SDIO bus configuration requests.

Handling SD/SDIO bus requests (Command / Response / Data).

Handling SDIO bus events (Card Detect, SDIO Interrupts, Request Completion) .
