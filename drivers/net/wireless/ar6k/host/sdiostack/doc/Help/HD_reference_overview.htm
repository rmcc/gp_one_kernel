Host Controller Driver Documentation


Abstract:


Host controller driver interfaces.

Configure The bus driver calls the configure callback to set various options and modes in the host controller hardware.
GET_CURRENT_REQUEST Get a pointer to the current bus request for a host controller
GET_SDCONFIG_CMD Get a pointer to the configuration command data.
GET_SDCONFIG_CMD_DATA Get a pointer to the configuration command data.
GET_SDCONFIG_CMD_LEN Get a pointer to the configuration command data.
IS_HCD_BUS_MODE_SPI Is host controller operating in SPI mode
IS_HCD_BUS_MODE_SPI_NO_CRC Is host controller using SPI in non-CRC mode
Request The bus driver calls the request callback to start an SDIO bus transaction.
SDHCD_GET_OPER_CLOCK Get host controller's current operational bus clock
SDIO_CheckResponse Check an SD/MMC/SDIO response.
SDIO_HandleHcdEvent Indicate to the SDIO bus driver (core) of an event in the host controller driver.
SDIO_RegisterHostController Register a host controller driver with the bus driver.
SDIO_UnregisterHostController Unregister a host controller driver with the bus driver.
