Codetelligence Embedded SDIO Stack
Support Functions Documentation


Abstract:


Debugging, synchronization and memory allocation.

CriticalSectionAcquire Acquire a critical section lock.
CriticalSectionDelete Cleanup a critical section object
CriticalSectionInit Initialize a critical section object.
CriticalSectionRelease Release a critical section lock.
DBG_ASSERT Evaluation the expression and throw an assertion if false.
DBG_PRINT Print a string to the debugger or console
KernelAlloc Allocate a block of kernel accessible memory
KernelFree Free a block of kernel accessible memory.
OSSleep Sleep or delay the execution context for a number of milliseconds.
REL_PRINT Print a string to the debugger or console
SDLIB_PrintBuffer Print a buffer on the standrad output
SemaphoreDelete Cleanup a semaphore object.
SemaphoreInitialize Initialize a semaphore object.
SemaphorePend Acquire the semaphore or pend if the resource is not available
SemaphorePendInterruptable Acquire the semaphore or pend if the resource is not available
SemaphorePost Post a semaphore.
