Embedded SDIO Driver Kit Help

Function Driver Interface


This section details the software interface between the SDIO function drivers and the bus driver (SDIO Core).

 

SDIO_RegisterFunctionRegister a function driver with the bus driver.
SDIO_UnregisterFunctionUnregister a function driver with the bus driver.
ProbeThis function is called by the Busdriver when a device is inserted that can be supported by this function driver.
RemoveThis function is called by the Busdriver when a device controlled by this function function driver is removed.
SDDEVICE_IS_SDIO_REV_GTEQ_1_10 Test the SDIO revision for greater than or equal to 1.10
SDDEVICE_IS_SD_REV_GTEQ_1_10 Test the SD revision for greater than or equal to 1.10
SDDEVICE_IS_MMC_REV_GTEQ_4_0 Test the MMC revision for greater than or equal to 4.0
SDDEVICE_IS_CARD_WP_ON Test for write protect enabled
SDDEVICE_GET_SDIO_FUNCNO Get the device's function number
SDDEVICE_GET_SDIO_FUNC_CLASS Get the functions's class
SDDEVICE_GET_SDIO_FUNC_CISPTR Get the functions's Card Information Structure pointer
SDDEVICE_GET_SDIO_FUNC_CSAPTR Get the functions's Code Stoarge Area pointer
SDDEVICE_GET_SDIO_FUNC_MAXBLKSIZE Get the functions's maximum reported block size
SDDEVICE_GET_SDIO_COMMON_CISPTR Get the common Card Information Structure pointer
SDDEVICE_GET_SDIO_CARD_CAPS Get the card capabilities
SDDEVICE_GET_CARD_FLAGS Get the card flags
SDDEVICE_GET_CARD_RCA Get the Relative Card Address register
SDDEVICE_GET_OPER_CLOCK Get operational bus clock
SDDEVICE_GET_MAX_CLOCK Get maximum bus clock
SDDEVICE_GET_OPER_BLOCK_LEN Get operational maximum block length.
SDDEVICE_GET_MAX_BLOCK_LEN Get maximum block length.
SDDEVICE_GET_OPER_BLOCKS Get operational maximum block count.
SDDEVICE_GET_MAX_BLOCKS Get maximum block count.
SDDEVICE_GET_CARDCSD Get the Card Specific Data Register.
SDDEVICE_GET_BUSWIDTH Get the bus width.
SDDEVICE_IS_BUSMODE_SPI Is bus in SPI mode.
SDDEVICE_CALL_REQUEST_FUNC Send a request to a device.
SDDEVICE_CALL_CONFIG_FUNC Send configuration to a device.
SDDeviceAllocRequest Allocate a request structure.
SDDeviceFreeRequest Free a request structure.
SDDEVICE_SET_IRQ_HANDLER Register an interrupt handler for a device.
SDDEVICE_SET_ASYNC_IRQ_HANDLER Register an asynchronous interrupt handler for a device.
SDDEVICE_GET_SDIOCARD_CAPS Get the SDIO capabilities register.
SDLIB_SetupCMD52RequestSetup cmd52 requests
SDLIB_IssueCMD52Issue a CMD52 to read or write a register
SDLIB_FindTuple Find a function tuple in common or function CIS.
SDLIB_IssueConfigIssue an SDIO configuration command.
SDLIB_SetFunctionBlockSizeSet function block size
SDLIB_PrintBuffer Print a buffer on the standard output
