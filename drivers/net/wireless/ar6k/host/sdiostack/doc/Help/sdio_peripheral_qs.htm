SDIO Function Driver Quick Start

The developer should first become familiar with the driver model for the target operating system. This manual provides only a brief overview of the driver development and debugging processes.

We encourage you to read the section: Developing Card Function Drivers. This SDK is supplied with a generic driver sample in the install tree located at \function\sample. The sample provides a UNIX-style streams interface for a console application to access function registers using CMD52 or CMD53. The sample driver demonstrates:

Function registration/deregistration.

Managing and tracking function instances.

Obtaining function specific information.

Issuing I/O commands (CMD52 and CMD53) synchronously.

Handling removal and cleanup.

Sample code for handling SDIO interrupts.
