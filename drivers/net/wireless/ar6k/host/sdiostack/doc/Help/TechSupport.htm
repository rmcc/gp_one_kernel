Embedded SDIO Driver Kit Help

Technical Support

The Embedded SDIO SDK includes sixty (90) days (from the time of purchase) of email based technical support. Technical support issues should be emailed to support@codetelligence.com.

Please include the following information when submitting your support email.

User Information Include name of company, name of user and software or registration key obtained from Codetelligence.
SDK Version

The version of the product can be found in this help file or the release notes.

Development Platform Specify the OS and version used for the development platform (Windows NT/XP, Redhat 8.0,9.0, Fedora 2,3)
Target Platform Specifiy the OS version of the target platform (Linux Kernel 2.x.x, QNX x.x, etc..)
Target CPU architecture Specify the target CPU architecture and platform : ARM (Gumstix), x86 (PC-Platform), etc..
Problem Description Provide a detailed description of the problem. Include step by step procedures to reproduce the problem.
