sdio_function.c

Abstract:

OS independent bus driver support for function drivers
This file supports the interface between SDIO function drivers and the bus driver.

_SDIO_RegisterFunctionRegister a function driver with the bus driver.
_SDIO_UnregisterFunctionUnregister a function driver with the bus driver.
Probe This function is called by the Busdriver when a device is inserted that can be supported by this function driver.
Remove This function is called by the Busdriver when a device controlled by this function function driver is removed.
